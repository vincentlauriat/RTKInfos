## features

Everything RTK saves, made visible

A tiny native menu bar app that reads RTK's local database and turns raw numbers into a glanceable picture of your savings.

📡

Real-time monitoring

Watches RTK's history.db via FSEvents, with a polling fallback — updates the moment a command runs.

📉

Compression gauge

The signature view: raw input compressed down to output, with the reclaimed tokens highlighted in emerald.

📊

7-day trend

Your daily savings, color-encoded by intensity, so you can spot the pattern at a glance.

🧾

Live Trace

A streaming side panel of your latest commands, newest first, each with its savings.

🚨

Status alerts

Detects a missing RTK install or more than 7 days of inactivity, and tells you.

⚙️

Preferences

Launch at login, configurable polling interval and a custom database path.

// why it exists

Numbers you'd otherwise never see

RTK does its work silently in the background. RTKInfos sits read-only on top of its database — a single @Observable source of truth driving a native SwiftUI dashboard — so the savings RTK earns you become tangible, day after day.

get started

Up and running in a minute

RTKInfos reads the database that RTK already keeps — so once RTK is set up, there's nothing else to configure.

Have RTK installed

RTKInfos visualizes RTK's data, so install and configure it first.

Download & install

Grab the latest .dmg and drag RTKInfos.app into /Applications.

Launch

Open it — the icon lands in your menu bar. Enable Launch at Login from Preferences.
